# 🚀 HabitFlow – Smart Habit Tracker

A modern and clean Habit Tracking Web App built using HTML, CSS, and JavaScript.

---

## 📌 About

HabitFlow helps you track your daily habits, visualize progress, and build consistency using a GitHub-style heatmap.

---

## 🎯 Features

* 📊 Dashboard (daily, weekly, monthly stats)
* 📅 Calendar view
* 🔥 Consistency Map (GitHub-style heatmap)
* ✅ Habit tracking system
* 📈 Reports & analytics
* 📝 Notes & Projects
* 🌗 Dynamic greeting system

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage

---

## ⚙️ How It Works

* Users add habits
* Track completion daily
* Data saved in browser (LocalStorage)
* Charts & heatmap update automatically
